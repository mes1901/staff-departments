## Installation

- Clone the repository git clone https://github.com/mes1901/staff-departments.git
- cd staff-departments
- composer install
- npm install
- npm run dev
- create database
- cp .env.example .env
- set DB connection configuration in .env
- php artisan config:cache
- php artisan migrate
- php ertisan db:seed
- php artisan serve
